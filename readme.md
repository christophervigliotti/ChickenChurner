# Chicken Churner

## Download This Code (The Right Way)

On MacOS if you don't have Git installed just type `git` into the terminal. 

Using Terminal...

1. navigate to the directory where you want to put the code files for this project
2. `git clone https://github.com/christophervigliotti/ChickenChurner.git .` < don't forget the dot

## Install Python Dependencies

`pip3 install sounddevice numpy scipy`
`pip3 install psutil`
`pip3 install librosa`

## Run
`python3 FILENAME.py` where FILENAME is the name of the file that you want to run
